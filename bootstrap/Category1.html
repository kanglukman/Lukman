Membuat Isi Halaman Website

Pembuatan isi halaman web Anda juga dapat dilakukan di dalam Fireworks CS4. Hal ini dikarenakan Fireworks CS4 memiliki kemampuan untuk mengimpor hasil desainnya ke dalam HTML, Image, dan CSS. Sehingga akan semakin memudahkan pekerjaan Anda nantinya dalam mengolah kembali halaman web tersebut dalam Dreamweaver CS4.

Ikuti contoh langkah-langkah untuk membuat isi halaman web berikut ini :

1. Sebagai contoh pertama adalah membuat isi halaman utama atau halaman index.html. Halaman ini adalah halaman yang akan tampil pertama kali saat Anda mengunjungi sebuah website.

Halaman index.html biasanya berisi informasi utama tentang sebuah website.

2. Buka kembali file index.png.

3. Klik tombol Text tool , arahkan pointer mouse ke dalam jendela dokumen dan buat sebuah kotak teks. Letakkan kotak teks tersebut di dalam kotak isi yang telah Anda buat pada latihan sebelumnya.

4. Ketik teks yang merupakan informasi utama tentang halaman web tersebut. Perhatikan contoh ilustrasinya seperti pada gambar di bawah ini.

5. Formatlah teks dalam kotak teks menggunakan panel Properties. Sebagai contoh ubah jenis huruf, ukuran, warna, dan perataan teksnya.

6. Setelah menambahkan teks utama, halaman index Anda semakin indah jika ditambahkan objek gambar ke dalamnya.

7. Pilih menu File â–º Import untuk mengambil gambar yang akan ditambahkan ke dalam jendela dokumen. Letakkan gambar-gambar tersebut hingga mendapatkan sebuah desain yang cantik.

8. Lakukan juga pengolahan warna gambar dengan menggunakan Blend Mode, tambahkan juga beberapa efek Live Filter agar gambar semakin indah.

9. Kembangkan sendiri kreasi Anda untuk mengolah isi halaman web Anda untuk mendapatkan hasil yang maksimal.
